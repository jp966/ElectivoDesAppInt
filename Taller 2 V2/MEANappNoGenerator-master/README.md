# AngularJS con NodeJS y MongoDB sin Generator
## Instalacion (eliminar node_modules y package-lock.json antes de seguir con los pasos)

```
npm install
```

## Iniciar servicio de MongoDB
```
sudo service mongod start
```

## Servir la aplicacion

```
npm start bin/www
```


## Ir a la pagina

```
localhost:3000
```
