Struts 2

Otro de los principales frameworks de Java utilizados por gran parte de los ingenieros de software más modernos es el sucesor de Apache Struts 1, su versión número dos. Este framework es todo un hallazgo para los desarrolladores que construyen nuevas aplicaciones web con Java EE . Apache ha entregado a estos un conjunto enorme de herramientas para crear aplicaciones web orientadas a empresas optimizando el proceso de desarrollo desde el inicio hasta el final y el mantenimiento posterior a la entrada en funcionamiento. Struts 2 es la elección correcta si la web va a tener una carga alta de datos, como puede ser una web de broadcast.
